# AI PowerBI Report Generator

AI-driven Power BI report generation — automate the creation of Power BI reports using natural language descriptions and AI models.

## Overview

This project aims to bridge the gap between natural language queries and Power BI report artifacts. Describe the report you want in plain English, and the AI generates the corresponding Power BI structure, measures, visuals, and layout.
